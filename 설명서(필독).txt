config
봇의 설정을 변경하는 폴더입니다. bot_info.json을 노트패드 혹은 메모장으로 여신 다음, 
봇 토큰을 바꾸고, owners 엔 자신의 유저 아이디를 넣으시면됩니다.
** 아이디 복사하는 방법 **
사용자 설정 => 디스플레이 => 하단에 개발자 모드 켜기 => 프로필 우클릭 => ID 복사하기

item
자판기 목록에 있는 파일들입니다.
!파일추가 이름 가격
으로 파일 DB를 생성하신 다음에,
"명령어에 썻던 이름과 똑같은 zip 형식" 의 압축파일을 넣으면됩니다.

예시 :

!파일추가 테스트 1000
(ITEM 폴더안)
테스트.zip

> 관리자 명령어
!충전 멘션/유저아이디 금액 
(멘션 혹은 유저아이디에게 금액 만큼 캐쉬를 충전합니다.)

!블랙추가 멘션/유저아이디
(멘션한 유저를 블랙으로 등록합니다. 등록 시 구매가 불가능해집니다.)

!블랙제거 멘션/유저아이디
(멘션한 유저를 블랙에서 제거합니다.)

!파일추가 파일이름 가격
(파일이름 파일을 가격으로 자판기에 올립니다.)

!파일제거 파일이름
(파일이름 파일을 자판기에서 제거합니다.)
